# WhatsApp Web Clone

Full stack WhatsApp Web clone.
Tech Stack and NPM Packages used -

1. react
2. firebase
3. material-ui
4. context api
5. react-router-dom

Clone the repo and do `npm start` to launch the application or checkout the [live demo here](https://whatsapp-clone-36b61.web.app/)
